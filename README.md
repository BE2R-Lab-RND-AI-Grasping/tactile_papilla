В репозитории представлен код генерации и обучения тактильных эластичных датчиков для роботизированных очувствленных схватов.

Для настройки камеры и параметров обучения - mini_camera.ipynb
Для анализа датасета - data_analysis.py
Для настройки выбора типоразмера датчика - sensor_picker_posthoc.py
Для запуска системы в инференсе - infernece.py
Для запуска роботизированной кисти с тактильными датчиками - hand_control.ipynb
